# Projeto Transportadora

O Projeto Transportadora é uma aplicação Java desktop para realização de controle fiscal de transporte de cargas.

## Tecnologias

As tecnologias envolvidas são as seguintes:
  * Java +8
  * JFrame
  * SQL Server
